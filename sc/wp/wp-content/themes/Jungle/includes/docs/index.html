Jungle

Multipurpose Responsive WordPress Theme

  • Created: 16/03/2013
  • latest Update: 04/04/2013

Jungle is a great, light, very flexible, option rich and responsive theme with many options to suit your business or personal needs.

Introduction

Jungle is a powerfull WordPress multipurpose theme that is very easy to set up and run a big or small site, literally from big corporate site to small business sites, personal or studio portfolios, photo albums, build for your idea to become true!

Installation

Requirements

Jungle theme requires WordPress 3.4 or greater. You must have WordPress already installed on your hosting. If you are not familiar of how to install WordPress please procede to this link "How to install WordPress"

Install the theme & activate it

Upload your theme:

1. Go to your WordPress admin dashboard, on the left panel navigate to Appearance > Themes

2. Click the Install Themes tab, and after that click on Upload link

3. Click on Choose File and select the already extracted jungle.zip file from the downloaded package

4. Click the Install Now button 

5. That's it, Jungle theme has been uploaded to your WordPress.

Activate your theme:

1. After confirmation of installation, navigate to Appearance > Themes > Manage Themes

2. Locate the Jungle theme and below the theme screenshot click on Activate

3. And Voila, now Jungle theme is activated and running.

Activate the premium plugins:

After activation you will be prompted to install the included premium plugins into your theme, the LayerSlider and the Symple Shortcodes plugins that are perfectly integrated to work flawlessly with our theme.

Installation video for Jungle Theme

Pages & Menus

After you have succesfully installed and activated Jungle theme now is the time to start populate it with pages and content for the pages, so let's make some pages and add content to them.

You can choose the following pages to display your desired content:

  • Default Template - It is the default page with sidebar on the right side.
  • Blog Medium Image - It is a template with blog layout with medium image.
  • Blog Full Width - It is a template page with full width image and content.
  • Full Width, No Sidebar - It is a template page for displaying static content on full width.
  • Portfolio One Column - Displays your portfolio items in one column with extra description or content, also buttons for the project and to visit external site.
  • Portfolio Two Column - DIsplays your portfolio items by two in each row.
  • Portfolio Three Columns - Displays your portfolio items by three in each row
  • Portfolio Three Columns with Sidebar - Displays your portfolio items three in row but with a sidebar option, the sidebar can be positioned right or left by choice.
  • Portfolio Four Columns - Displays your portfolio items by four in each row

Pages

Create a New Page:

1. Go to the left panel and navigate to Pages > Add New

2. Enter the title for the new page, for e.g Home Page

Choose a page template for your page:

Under page attributes you can select under what template this page to be displayed, or simply what layout to use.

Page Options:

Under page options you can choose to:

1. Add custom title for the page (different from the already existant title) or

2. Choose to display or not the title (for example you don't want or need to display the page title or the custom title at all)

LayerSlider Options:

We have made it easy for you to include different layerslider on each separate page, first of all to use this feature you must already have made a slider with the LayerSlider plugin:

1. Enable or Disable the slider (every time you create a new page, the layerslider is disabled by default)

2. Choose which slider you want to display on the page by simply putting it's ID number

3. Choose and select a color for the background of the slider.

 

Menus

Let's create our first menu and add the above created page to it. 

Jungle theme has one primary menu has one location that is displayed in the header of the theme.

1. Navigate under Appearance > Menus

2. Add a name for your new menu in the input field labeled Menu Name and click Create Menu.

3. Under Theme Locations on the left side select the Primary

3. On the left side under Pages you will notice your already created page named Home Page, check it and click Add to Menu

4. Great, now you have your new menu and it is displayed on the frontend.

5. You can select and add pages, categories, links to the menu, they will be succesfully displayed on the frontend.

Setting the contact form

If you want to display the contact form exactly as is in the demo, than download the contact form 7 plugin, install it, after instalation open the contact form and add the below code into the form field:


	<fieldset class="small-input">
		<label>Name (required)</label>[text* your-name]
	</fieldset>

	<fieldset class="small-input">
		<label>Email (required)</label>[email* your-email]
	</fieldset>

	<fieldset class="small-input last">
		<label>Website</label>[text* your-website]
	</fieldset>

	<fieldset>
		<label>Your comment (required)</label>[textarea your-message]
	</fieldset>
	[submit class:btn "Send Message"]

- Twitter Widget

This widget grabs your or others accounts latest tweets and displays them. It uses the new twitter API version 1.1, that means that it is obligatory for you to have the following credentials so the widget can work properly. To get those api credentials you have to open a new account at the following link https://dev.twitter.com/ , after you have registered or signed it with your already created twitter account, login and create a new twitter application, after that they will give you the following credentials, they are:

- Consumer Key

- Consumer Secret

- Access Token

- Access Token Secret

Add all of them into the widget inputs, choose from which twitter username you want to grab tweets and how many of them to be displayed and you are ready to go!

Translation

Jungle theme is a translation ready, that means you can easily translate to your native language.

You will need the POEdit tool to translate all the strings from Jungle theme, you can download the software from HERE,  after you have downloaded and installed the software, open the english.po file from the includes/languages folder and translate the texts.

Save it in the same folder with a new name or simply overwrite the english one (only if you don't need the english language). If you save it under new name let's say you translate it to german and you saved it as de_DE, than go to the root of your WordPress installation and open the wp-config.php file find the line with define('WPLANG', ''); and modify it like this define('WPLANG', 'de_DE'); 

And that's it, the Jungle theme is translated.

How to Update Jungle Theme?

Immediately after download, please go to your themeforest profile, and click on the Downloads tab, this is where you can download Jungle, now for you to always get the latest updates on time, please click on the thickbox where it says "Get notified by email if this item is updated" as in the screenshot below.

After you have done that, click the download button and download the latest version of Jungle.

Update to current or latest version

VERY IMPORTANT: Always before you do the update, backup your theme.

NOTE: If you have changed the theme core files by yourself, or someone else did it for you, or simply you made some small custom modifications, make sure you know where they are and check the update log to see if those files were affected with the update.

There are two ways of updating your theme:

1. Upload the new theme folder files via FTP.
2. Delete the theme from the WordPress via the admin, and upload the new version (delete because to prevent the folder exists error)
